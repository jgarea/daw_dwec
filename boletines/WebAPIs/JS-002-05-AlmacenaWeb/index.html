almacenamiento Web

Este ejemplo está diseñado para demostrar el uso de la W3C Web Storage API. Debería funcionar desde IE8. Elije colores de fondo, logotipos y fuentes personalizados de los menús desplegables a continuación, luego cierra y vuelve a abrir la página; encontrarás que tus elecciones se recuerdan, ya que se almacenan mediante el almacenamiento web. También puedes visitar la salida de eventos de almacenamiento (se abre en una pestaña nueva). Ábrelo y cambia algunos valores en la página de índice, luego mira la de eventos: verás los cambios de almacenamiento informados.
